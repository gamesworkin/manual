GUIA DE INICIALIZAÇÃO

Sistema OPL & FMCB 1.8 (Funtuna)

Este manual mostra dois métodos para poder carregar seus jogos diretamente pelo OPL utilizando o sistema funtuna. Escolha o método que melhor se adapta à configuração de seu PS2.

MODO 1: Inicialização direta pelo OPL (Slot 1)

  1. Insira o Memory Card preparado no Slot 1 do seu PlayStation 2.
  2. Conecte o seu pendrive ou HD externo em uma das duas portas USB do seu PS2.
  3. Pressione e mantenha pressionado o botão R2 no controle.
  4. Ligue o PlayStation 2 mantendo o botão R2 pressionado até que a tela comece a piscar e o OPL apareça e abra completamente.
PLATAFORMA BÔNUS

MODO 2: Inicialização manual pelo FreeMCBoot (Slot 1)

  1. Insira o Memory Card no Slot 1 do seu PlayStation 2.
  2. ATENÇÃO! Não conecte o pendrive neste momento.
  3. Pressione e mantenha pressionado o botão R1 no controle.
  4. Aguarde alguns instantes até que o MENU do FreeMCBoot carregue na tela.
  5. Agora, conecte o seu pendrive.
  6. Selecione a opção Open PS2 Loader (OPL) para iniciar a lista de jogos.
